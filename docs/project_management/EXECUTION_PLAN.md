# WSIS Execution Plan

## Agent Responsibilities

### 1. Investigation Agent

Goal: understand what the project currently builds.

Deliverables:

- Current architecture summary
- Data-source inventory
- UI flow inventory
- GitHub/ClickUp status summary
- David/Sarah gap report

### 2. Inspection Agent

Goal: verify the current build and identify defects.

Deliverables:

- Test results
- API response spot checks
- Dataset confidence audit
- UI claim audit
- Technical vulnerability list

### 3. Builder Agent

Goal: implement the reworked product.

Deliverables:

- David decision engine
- Sarah skeptic review engine
- Verdict-first UI
- Evidence disclosure views
- Unit and integration tests

### 4. Responsive QA Agent

Goal: verify mobile usability.

Deliverables:

- iPhone viewport screenshots
- Touch target review
- Text overflow review
- Mobile table-to-card review
- Duplicate-content review
