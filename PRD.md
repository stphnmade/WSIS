# WSIS — Where Should I Start

Product Requirements Document

WSIS is an interactive decision tool that helps young professionals evaluate cities in the United States based on affordability, lifestyle, and social sentiment.

Core idea:
Data tells you what a city costs.
People tell you what it feels like.

Primary Features
- Interactive livability map
- City scoring engine
- Reddit social sentiment panel
- City comparison
- Watchlist (OAuth optional)

Tech Stack
Python
Streamlit
Plotly
Geopandas
Mapbox
FastAPI
AWS Lambda
Docker
Supabase/Postgres
